Customer data generator :param customer_number: The number of customers. Defaults to 10. :type customer_number: int, optional :param seed: A random seed for faker. Defaults to 4444. :type seed: int, optional :param profile_path: Customer Profile path. Defaults to “./profiles/main_config.json”. :type profile_path: str, optional :param file_path: Output path to store the dataset in csv format. Defaults to “./data/customers.csv”. :type file_path: str, optional

Returns: pandas.DataFrame(): The customer data set in pandas frame format. Specially, if customer_number==1, the generator just generates one customer data.
